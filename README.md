# autonomous-follow-drone

## Description

A DIY autonomous drone capable of following a user using computer vision-based tracking. The project demonstrates robotics principles, embedded systems, and autonomous control.

## Features

- basic drone flight
- computer vision tracking (in progress)
- autonomous follow capability (planned)
- modular design for future expansion

## Hardware

- quadcopter frame
- flight controller
- brushless motors
- Raspberry Pi (vision processing)
- camera module

## Software

- Python
- OpenCV
- vision tracking (planned)
- control algorithms (planned)
